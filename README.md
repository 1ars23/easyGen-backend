## Description

NestJS Login Signup repository.

## Project Steps
#### Step 1

```bash
$ npm install
```

#### Step 2

```bash
# Setup Environment File
$ cp .env.example .env
```

#### Step 3

```bash
# Docker Setup
$ docker-compose up --build -d
```

#### Swagger
```bash
# Access Swagger Documentation
# Swagger is available at: 
$ http://localhost:<port>/api-docs

```
